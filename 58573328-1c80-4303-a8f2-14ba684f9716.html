Description

Couch Goblins is a 4 player local multiplayer party game. This demo features 3 obstacle courses that can be played solo, co-operatively or competitively. Future demos will feature eccentric mini games, PvPvE arena modes, and more! 

  • Ages: 6+ 
  • Players: 1-4 
  • Gamepad Compatible 
  • System Requirements: Win64 system with a dedicated graphics card recommended.  

Known Bug:
Press Windows Key and click back into game if you cannot access Start Menu. 

Optional Keyboard/Mouse Controls (Player 1):

  • Attack = Mouse1
  • Jump = Space
  • Shift = Sprint
  • F = Interact/Zipline


Developed during the Summer 2021 NA PUBG Hackathon.
